# Контекст проекта (передача в Claude Code)

Этот файл описывает, что за проект, как он устроен и что планировалось дальше. Можно удалить после прочтения.

## Что это

Мобильная словесная игра «Метаграммы» (она же word ladder / «дублеты» Льюиса Кэрролла). Игрок превращает одно слово в другое, меняя по одной букве за ход; каждое промежуточное слово должно быть реальным.

Чистый статический фронтенд: HTML + CSS + JS, без сборки и фреймворков. Иконки — Tabler Icons через CDN.

## Структура

- `index.html` — разметка
- `style.css` — стили, светлая/тёмная тема через prefers-color-scheme, адаптив под маленькие экраны
- `script.js` — вся логика: набор цепочек (PUZZLES), словарь (WORD_DICT), рендер лестницы, экранная клавиатура, подсказки
- `README.md`, `LICENSE`, `.gitignore`

## Ключевые решения

1. **Вся лестница видна сразу.** Между стартом и целью заранее показаны пустые ячейки по числу шагов эталонного решения — чтобы игрок понимал масштаб задачи. Это сделано осознанно по просьбе заказчика.

2. **Цепочки выверены вручную.** Взяты из проверенного списка метаграмм (файл METAGRAMMY.docx). Ранее были самопальные цепочки с ошибками (несуществующие слова, переходы с заменой двух букв) — они полностью заменены. При добавлении новых цепочек ОБЯЗАТЕЛЬНО проверять: каждое слово реально существует, каждый переход меняет ровно одну букву, все слова одной длины.

3. **Словарь пока минимальный.** WORD_DICT строится из объединения всех слов всех цепочек (~130 слов). Это главное ограничение: альтернативные корректные пути игра не примет.

## Что планировалось дальше (приоритет сверху вниз)

1. **Расширить словарь** — подключить OpenCorpora или Mystem, отфильтровать существительные в именительном падеже единственного числа (~5–10 тыс. слов), сохранить как dict.txt и подгружать. Это снимет главное ограничение.
2. **Генератор уровней** через BFS по графу слов (рёбра = замена одной буквы).
3. **Ежедневная общая загадка** для всех (как Wordle) + шеринг результата эмодзи-сеткой.
4. Анимация перехода между строками, хаптика, подсветка использованных букв на клавиатуре.

## Известные мелочи в исходном списке метаграмм

- Цепочка `ПОРТ → ПОСТ → ПОЭТ → ПОРТ → ПОРА` повторяет ПОРТ дважды — вероятно опечатка, в текущий набор не включена.
- `ЛАЙКА → ЛАСКА` в исходнике дублировалась — оставлена одна.

## Запуск

Открыть index.html в браузере, либо `python3 -m http.server 8000`.
