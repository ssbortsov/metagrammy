# Метаграммы

Мобильная словесная игра. Преврати одно слово в другое, меняя по одной букве за ход. Каждое промежуточное слово должно быть реальным.

Жанр известен с XIX века как «дублеты» Льюиса Кэрролла, в русской традиции — «метаграммы».

## Демо

Открой `index.html` в браузере — игра работает локально без сервера и без сборки. Достаточно дабл-клика по файлу.

Для публикации на GitHub Pages см. раздел ниже.

## Что внутри

- `index.html` — разметка
- `style.css` — стили (со светлой и тёмной темой через `prefers-color-scheme`)
- `script.js` — логика игры, словарь и набор уровней
- иконки — Tabler Icons через CDN

Зависимостей нет, фреймворков нет, сборки нет.

## Механика

Каждый уровень — пара слов «из чего» и «во что». Между ними нужно построить лестницу промежуточных слов, заменяя по одной букве за шаг. Все слова должны быть в словаре игры.

Лестница показана сразу: пустые ячейки внизу заранее обозначают количество шагов до цели, чтобы было видно масштаб задачи.

Управление: тап по ячейке выбирает позицию для замены, экранная клавиатура вводит букву, кнопка OK подтверждает ход. На десктопе работает физическая клавиатура.

## Текущие ограничения

Словарь сейчас минимальный — только слова, встречающиеся в 26 заложенных цепочках. Это значит, что альтернативные пути к цели игра не принимает, даже если они корректные. Для продакшена нужен большой словарь русских нарицательных существительных в именительном падеже единственного числа (примерно 5–10 тыс. слов).

Подсказка работает только по эталонному пути. Если игрок ушёл в сторону — игра честно об этом говорит и предлагает идти своим путём.

## Развитие

- Подключить словарь OpenCorpora с фильтрацией по части речи и грамматической форме
- Генератор уровней через BFS по графу слов
- Ежедневная общая загадка для всех игроков (как Wordle)
- Шеринг результата эмодзи-сеткой
- Анимация перехода между строками
- Хаптика на победу и ошибку
- Подсветка использованных букв на клавиатуре

## Запуск локально

```bash
# Никакой установки — просто открой index.html
open index.html

# Или через любой статик-сервер
python3 -m http.server 8000
# затем http://localhost:8000
```

## Публикация на GitHub Pages

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main
```

Затем в настройках репозитория: **Settings → Pages → Source: Deploy from a branch → main / root**. Через минуту игра будет доступна по `https://USERNAME.github.io/REPO/`.
